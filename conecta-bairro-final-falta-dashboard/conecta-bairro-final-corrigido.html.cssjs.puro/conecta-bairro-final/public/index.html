Sobre o Conecta-Bairro

O Conecta-Bairro é uma plataforma web desenvolvida como projeto de extensão universitária na disciplina Programação para Web, com o objetivo de fortalecer a economia local de Águas Claras, Brasília.

Dados de Águas Claras

  • População: 141.872 habitantes
  • Densidade: 14.074 pessoas/km² (maior do DF)
  • Renda média: 18 salários mínimos
  • Crescimento: ~8% ao ano
