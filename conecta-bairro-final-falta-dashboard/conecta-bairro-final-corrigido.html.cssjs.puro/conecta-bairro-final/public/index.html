QUEM SOMOS

Sobre o Conecta-Bairro

Uma plataforma web criada como projeto de extensão universitária na disciplina de Programação para Web, com um propósito simples: fortalecer a economia local de Águas Claras conectando moradores a prestadores de serviço de confiança do próprio bairro.

🤝

Sem intermediários caros e sem complicação — só vizinhos ajudando vizinhos, de forma simples, transparente e humana.
